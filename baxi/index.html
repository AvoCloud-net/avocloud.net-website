ABOUT BAXI

What is Baxi?

Baxi is a modern, privacy-focused Discord bot designed to make server management safer, smarter, and easier. It combines powerful moderation tools, an AI-powered chat filter, cross-server global chat, and much more — all configurable through an easy-to-use web dashboard.

Baxi is part of the AvoCloud project, which focuses on creating modern, privacy-friendly tools for Discord communities.

FEATURES

What Baxi Offers

Moderation

Full moderation toolkit with slash commands for ban, kick, unban, mute, warn, and bulk message deletion (/clear). Bans and mutes can be set as temporary with a custom duration — Baxi lifts them automatically when the time is up and notifies the user via DM if possible. The warning system auto-escalates: reaching configured thresholds triggers a mute, then kick, then ban — automatically, without manual intervention.

Anti-Spam

Detects message spam, mention floods, and repeated content in real time. When a threshold is exceeded, Baxi automatically takes action — configurable between warn, mute, kick, or ban. Keeps your server safe without manual intervention.

AI-Powered Chat Filter

Two-tier filtering system to keep your server clean. SafeText handles open-source bad-word detection, while the AI Filter uses an LLM (Llama Guard) to analyze message context — catching hate speech, NSFW content, and self-harm references that simple word lists miss. Flagged messages are automatically deleted. A bypass channel can be configured per server.

Global Chat

A cross-server chatroom that connects all Discord servers with Global Chat enabled — messages including attachments and replies are relayed to every connected server. Every message is always checked by Baxi's AI Filter, and a dedicated moderation team monitors the chat around the clock. Only available on the official hosted bot.

Ticket System

Members open support tickets via a modal — choosing a custom title and describing their issue at creation. Staff can claim, close, and archive tickets as transcripts. A configurable staff role gets access to all tickets, keeping requests out of public channels.

Custom Commands

Define your own text commands with a custom trigger and response — no coding required. Use any prefix and keyword you like (e.g. !help, .invite). Variables like {user}, {server}, and {membercount} make replies dynamic, with optional embed support. Perfect for automating FAQs, server rules, or recurring announcements.

Welcome & Leave Messages

Greet new members with a fully customizable welcome message using variables (username, server name, etc.). Baxi can optionally generate a welcome image with a custom background to match your server's style. The goodbye message is configurable separately.

Auto-Roles

Automatically assign one or more roles to every user when they join the server. Set it up once in the dashboard — no further action needed. Useful for default member roles, bot separators, or any role that should apply to all new members.

Twitch Live Tracking

Track Twitch channels and automatically post a live announcement when a stream starts — showing stream title, viewer count, and a preview image. The message updates live so there's no spam of new posts. The channel name changes to 🔴 live / ⚫ offline, and an optional role ping notifies your community instantly.

Stats Channels

Voice channels that automatically display live server statistics — total members, humans, bots, channels, and roles. Each stat gets its own channel with a fully customizable name template, updated automatically every 10 minutes.

Global User Flagging

A shared database of flagged users across all Baxi servers. Use /scan_users to check your server for reported members and take action proactively — before issues arise. Helps communities stay safe from known bad actors across the network. Only available on the official hosted bot.

Web Dashboard

Configure every Baxi feature through an intuitive web interface — login with Discord OAuth2, no slash commands or config files needed. Manage chat filter settings, welcome messages, custom commands, Twitch streamers, auto-roles, and more, all in one place. Includes an audit log and feature adoption statistics. Changes apply to your server instantly.

Installation & Setup

Official Hosted Bot (Recommended)

Get started immediately with the official Baxi bot. All features work out of the box.

Invite Bot ↗ Dashboard ↗

Self-Hosting

Clone and install Baxi:

git clone https://github.com/AvoCloud-net/Baxi.git
cd Baxi
pip install -r requirements.txt

Configure config/auth.py with your Bot Token, then run: 

python main.py
GitHub Repository ↗

Self-Hosting Limitations

Global Chat

Not available (requires official Baxi network access)

SafeText Filter

Requires a Public API Key (request from AvoCloud)

AI Filter

Must be self-hosted (recommended: OpenWebUI + Ollama with Llama-Guard 3)

Other Features

Work fully without restrictions

PRIVACY

Privacy Policy

Minimal Data Collection

Baxi follows a minimal data collection principle. All features are optional — users decide what to enable and what data (if any) is stored.

✓ Stored Data

  • • Chat Filter Logs (only if enabled, only for removed messages)
  • • Server Configuration (dashboard settings)
  • • Global Chat Images (for cross-server delivery)

✗ NOT Stored

  • • Other messages
  • • Personal data
  • • Private information
  • • Tracking data
