ABOUT BAXI

What is Baxi?

Baxi is a modern, privacy-focused Discord bot designed to make server management safer, smarter, and easier. It combines powerful moderation tools, a customizable verification system, and global chat functionality, all configurable through an easy-to-use web dashboard.

Baxi is part of the AvoCloud project, which focuses on creating modern, privacy-friendly tools for Discord communities.

FEATURES

What Baxi Offers

AI-Powered Chat Filter

Two-tier filtering system: SafeText (open-source bad-word detection) + AI Filter (LLM-based content analysis hosted on AvoCloud servers).

Global Chat

Inter-server chatroom connecting multiple Discord servers. (Only available on official hosted bot)

Flexible Verification

Customizable onboarding: Button only, Button + CAPTCHA, or Button + Password.

Ticket System

Support ticket creation for help requests or moderation reports.

Web Dashboard

Full configuration of all features via a web interface — no complex commands needed. Everything is just a few clicks away.

GET STARTED

Installation & Setup

Official Hosted Bot (Recommended)

Get started immediately with the official Baxi bot. All features work out of the box.

Invite Bot ↗ Dashboard ↗

Self-Hosting

Clone and install Baxi:

git clone https://github.com/AvoCloud-net/Baxi.git
cd Baxi
pip install -r requirements.txt

Configure config/auth.py with your Bot Token, then run: 

python main.py
GitHub Repository ↗

Self-Hosting Limitations

Global Chat

Not available (requires official Baxi network access)

SafeText Filter

Requires a Public API Key (request from AvoCloud)

AI Filter

Must be self-hosted (recommended: OpenWebUI + Ollama with Llama-Guard 3)

Other Features

Work fully without restrictions

PRIVACY

Privacy Policy

Minimal Data Collection

Baxi follows a minimal data collection principle. All features are optional — users decide what to enable and what data (if any) is stored.

✓ Stored Data

  • • Chat Filter Logs (only if enabled, only for removed messages)
  • • Server Configuration (dashboard settings)
  • • Global Chat Images (for cross-server delivery)

✗ NOT Stored

  • • Other messages
  • • Personal data
  • • Private information
  • • Tracking data

Slash Commands
