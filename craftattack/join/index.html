How to Join — Craft Attack Community Server

Step-by-step guide to join the open Avocloud Craft Attack community server, Season 2: The Ashes of Aerlon. No application form. No fee. No closed cast.

Season start: Thursday, May 14, 2026 — 17:30 Europe/Vienna.
Server address: craftattack.avocloud.net
Editions: Minecraft Java + Bedrock (crossplay)
Cost: Free

Join in 4 Steps

  1. Join the Avocloud Discord
    Open avocloud.net/discord ↗ in your browser and join the server.
  2. Connect to craftattack.avocloud.net in Minecraft
    First connection won't let you in yet — instead the server shows a one-time link code. Copy it. Works for Java and Bedrock.
  3. Run /mc link <code> in Discord with Baxi
    In our Discord, use the Baxi bot command /mc link and paste the code from step 2. Baxi will reply with a confirmation message once the link is registered.
  4. Reconnect to the server
    After Baxi confirms, reconnect to craftattack.avocloud.net. You are now whitelisted and inside. That's it.

FAQ

Do I need a paid Minecraft account?
Yes — a legitimate Minecraft Java or Bedrock account. No cracked clients.

Can I play on console / mobile?
Yes. Bedrock Edition supports Windows 10/11, Xbox, PlayStation, Switch, iOS, Android.

Is there an application form?
No. Linking Discord + Minecraft via Baxi is all you need.

Can I get banned for the wrong mod?
See Server Rules — performance mods (Sodium, Iris, OptiFine), shaders, and surface minimaps are fine. X-Ray, cave-mode minimaps, entity radar, killaura etc. are banned.
