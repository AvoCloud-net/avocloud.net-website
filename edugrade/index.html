ABOUT

What is EduGrade?

EduGrade is a secure web application designed for teachers and educators to manage student grades, classes, and performance tracking. Built with privacy and security in mind, it features AES-256-GCM encryption to protect all sensitive student data.

Whether you're tracking grades for a single class or multiple subjects, EduGrade provides powerful analytics, customizable grading systems, and secure grade sharing with students — all wrapped in a clean, responsive interface that works on any device.

SECURITY

Built with Security in Mind

AES-256-GCM Encryption

All user data is encrypted at rest using AES-256-GCM encryption with PBKDF2 key derivation (200k iterations for passwords, 100k for encryption keys). The zero-knowledge architecture means server admins cannot read data without the password.

AES-256-GCM PBKDF2 Zero-Knowledge Encrypted at Rest

Short Sessions

1-hour sessions with automatic cleanup ensure your data stays protected even on shared devices.

1-Hour Sessions Auto Cleanup

Protected Sharing

6-digit PINs per student with encrypted grade shares and automatic expiration cleanup.

6-Digit PIN Encrypted Shares Auto Expiry

DEPLOYMENT

Get Started with EduGrade

Docker (Recommended)

Quick deployment using Docker:

docker pull redwolf2467/edugrade
docker run -d --name edugrade -p 8080:1601 \
  -v edugrade-data:/app/data \
  --restart unless-stopped \
  redwolf2467/edugrade:latest
Docker Hub ↗

Docker Compose

Use the provided docker-compose.yml configuration file from the repository for easy deployment with persistent storage.

Persistent Storage Easy Setup

Manual Installation

Install from source:

git clone https://github.com/rwolf2467/edugrade.git
cd edugrade
pip install -r requirements.txt
python app.py
